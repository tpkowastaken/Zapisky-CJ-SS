# Zapisky-CJ-SS
Zápisky z Češtiny na střední škole od prváku až do čtvrťáku pomocí GFM (Github Flavored Markdown)

# Format
```
# Témata
## Skupiny/Národnosti
### Autoři

Nadpisy jsou vždy bez bulletů
* První bullet
  * Druhý bullet
    * Třetí bullet...
```
jak to vypadá můžete vidět v example.md
